# 26.6.0 - Initial Release

**Released:** June 3, 2026

## Added

- Initial project scaffold with Vue 3 frontend powered by Vite
- FastAPI backend with SQLModel ORM
- VitePress documentation site
- Configuration Items management
- L2 and L3 network role definitions
- VLAN, MAC address, and IP prefix management
- Autonomous System and RIR data models
- GitHub Actions workflow for docs deployment to GitHub Pages
- Custom domain support (assets.mcguire.technology)

## Architecture

- **Frontend:** Vue 3 + TypeScript + Tailwind CSS
- **Backend:** FastAPI + SQLModel + SQLite
- **Docs:** VitePress

## Known Limitations

- SQLite database (suitable for development; production should migrate to PostgreSQL)
- No authentication or RBAC implemented
- Sample/placeholder data only

## Next Steps

- Add authentication and role-based access control
- Migrate to PostgreSQL for production
- Expand data models and API endpoints
- Add comprehensive test coverage
