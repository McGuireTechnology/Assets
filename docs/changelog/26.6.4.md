# 26.6.4 - DigitalOcean Backend Deployment Prep

**Released:** June 4, 2026

## Added

- Add a Dockerfile for running the FastAPI backend on DigitalOcean App Platform
- Add a DigitalOcean app spec for the backend API and managed PostgreSQL service
- Add deployment documentation for GitHub Pages, Cloudflare Pages, and DigitalOcean
- Add a Mise task for building the backend Docker image locally

## Changed

- Configure the backend to read CORS origins, app version, and database URLs from deployment environment variables
- Normalize DigitalOcean PostgreSQL URLs to use the `psycopg` SQLAlchemy driver
- Include the app version from the root `VERSION` file in FastAPI metadata

## Fixed

- Add tests for environment parsing and PostgreSQL URL normalization
